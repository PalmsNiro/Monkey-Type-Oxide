# Monkey-Type-Oxide

## General
A clone of MonkeyType made with rust and ratatui for the terminal

## Controls
| Control       | Keybind 1     | Keybind 2      |
| ------------- | ------------- | -------------- |
| next Tab      | strg + l      | strg + Arrow r |
| previous Tab  | strg + h      | strg + Arrow l |
| quit          | Esc           | strg + q       |
